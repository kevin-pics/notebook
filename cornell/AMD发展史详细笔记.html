🌟 第一章：半导体行业的起源

🔬 贝尔实验室与晶体管的诞生

1947年 - 三位科学家制造出全世界第一个晶体管 - 获得诺贝尔物理学奖 - 其中一人：威廉·肖克利 (技术大宗师,但管理能力差)

🏢 仙童半导体 - 硅谷的起源

1957年 - "八叛徒"的出走

八位顶尖科学家因不满肖克利的管理,集体离开创立 Fairchild Semiconductor (仙童半导体)

关键地位: - ✨ 半导体领域的人才摇篮 - 🌆 可以说是硅谷的起源 - 📤 十年间创始人纷纷出走,孵化无数公司

🎯 两大巨头的诞生

1968年 - 英特尔成立 - 创始人: 诺伊斯 & 摩尔 (仙童最后离开的两位) - 融资: 轻松融到 250万美元 - 两年后即IPO上市

1969年 - AMD成立
- 创始人: 桑德斯 (仙童销售主管) - 资金: 💸 第一笔投资竟来自英特尔CEO诺伊斯! - 1972年 上市

💡 核心观察: AMD和英特尔从诞生之日起就紧密纠葛在一起

🚴 第二章: 早期战略 - "Drafting"跟车战术

🤝 第二供应商策略 (1970-1990s)

背景: - 美国厂商采购芯片时要求第二供应商,防止被卡脖子 - AMD经常作为英特尔的第二供应商

策略比喻: 骑行中的Drafting 领头羊(英特尔) → 减小风阻 → 跟随者(AMD) ↓ 双方都省力 ↓ 形成小车队

🎊 最成功的合作: IBM订单

1980s - 拿下IBM个人电脑市场 - 英特尔: x86架构成为CPU行业标准 - 两家公司一起起飞 - 英特尔 2000年市值: 2750亿美元 (全球第6) - AMD 五年股价: 📈 翻了15倍

⚠️ 关系破裂

  • 个人电脑市场逐渐饱和
  • 出现授权方面的诉讼
  • AMD意识到: 不能永远跟在英特尔后面!

🔥 第三章: 第一次逆袭 - 速龙时代 (1995-2006)

💪 自立门户的准备

1995年 - 战略转型 - 大幅提高研发费用 - 💰 花 9亿美元 收购 NexGen 半导体设计公司

关键人物: - 达姆 - NexGen的COO - 身份: 😱 奔腾处理器之父 (刚从英特尔挖来!) - 鬼使神差加入AMD

🚀 K系列架构横空出世

1996年 - 推出自研 K系列架构

1999年6月 - 搭载K7架构的 速龙 (Athlon) 发布

1999年 - 不到10个月后 - ⚡ 发布全球首款 >1GHz 时钟频率处理器 - 正面对抗英特尔 奔腾III - 性价比领先一大截!

📈 业绩大翻身

年份 净利润
1998 ❌ 亏损 8900万美元
1999 ❌ 亏损近 1亿美元
2000 ✅ 盈利 9.8亿美元 🎉

💬 "这个速龙,它就是这么速度!"

🏆 巅峰时刻

2003年 - 推出搭载K8架构的首款 64位 x86处理器

2006年市场份额: - 整体CPU市场: 48.4% (几乎平分天下!) - 台式机: 📊 超过英特尔

💔 第四章: 收购ATI - 黑暗十年的开始 (2006-2014)

🎯 野心勃勃的战略

核心思路:

融合CPU + GPU → 横跨两大芯片领域 → 干翻英特尔!

2000年市场格局: - CPU市场: 英特尔 vs AMD
- GPU市场: 英伟达 vs ATI

🤔 收购对象的选择

选项分析:

公司 市值 分析
英伟达 75亿美元 AMD才86亿,太接近,可能反被吞 😰
ATI 50亿美元 有一定距离,收购可行 ✅

💸 豪赌式收购

收购成本: - 现金: 18亿美元 (掏空全部) - 借债: 25亿美元
- 股票: 11亿美元 - 总计: 54亿美元

⚠️ 豁了半条老命!

结果: - 成为唯一在CPU和GPU两大市场都有相当份额的巨头 - 听起来很厉害...对吧?

😱 现实的残酷打击

两年后的账面调整: - 商誉减记: -26亿美元 - 意味着: 😭 花54亿买的东西,只值20多亿!

问题分析: 1. ✅ 战略方向非常正确 2. ❌ 但买太贵了,严重影响现金流 3. 🔥 同时英特尔开始反击

⚔️ 英特尔的强势反击

实力对比的真相: - AMD营收: X - 英特尔营收: 10X (十倍差距!) - 英特尔市值: 30X (三十倍差距!)

英特尔的"Tick-Tock"战术: Tick年 → 更新芯片制程 Tock年 → 更新处理器架构 结果: 自己卷自己,一路领先

2006年 - 收购ATI宣布后第3天 - 英特尔发布 酷睿2处理器 (全新Core微架构) - 性能 & 能效比都非常出色 - 重新拉开与AMD的差距

📉 市场份额大崩盘

CPU市场份额: - 高光时刻: 50%
- 跌至: 20% 😢

服务器CPU市场份额: - 曾经: 30% - 跌至: 1% 😱

市值暴跌: - 跌至 不到20亿美元 - 活下来都成问题!

💔 绝望的自救

2008年 - 被迫出售晶圆厂 - 拆分出 GlobalFoundries - 目的: 换钱 + 专注芯片设计

其他措施: - 不停裁员 - 削减成本
- 😭 甚至卖掉总部大楼 (换1.6亿美元)

💬 华尔街分析师: "AMD可能要退出历史舞台了"

👑 第五章: 苏妈时代 - 浴火重生 (2014-)

🌈 救世主降临

2014年 - 苏姿丰 (Lisa Su) 出任CEO

背景: - 🎓 MIT电气工程博士 - 💼 德州仪器、IBM工作经历 - 🔧 芯片技术精英

🎵 "驾着七彩祥云,伴随着背景音乐从天而降"

有趣观察: - 苏妈和老黄(英伟达CEO)都语速超快 📢 - 正式场合还控制一下 - 私下采访真的像机关枪 😄

🎯 两大战略转变 (地狱级难度)

战略一: 🚀 高性能计算 (High Performance Computing)

核心理念:

"中低端没前途,利润率太低!" - 苏妈

问题: - AMD之前主打性价比,集中在中低端 - 虽然市场份额高,但利润率低

解决方案: - 重新聚焦高性能芯片开发 - 苏妈口头禅: "High performance computing"

战略二: 📈 拓展业务线

2015年提出三大增长领域: 1. 🎮 游戏 2. 🏢 数据中心
3. 🥽 沉浸式平台

💡 现在看来,这眼光真的很毒辣!

⚡ Zen架构 - 反击的号角

2016年底 - 推出搭载 Zen架构 的锐龙芯片

技术特点: - 使用 Chiplet (小芯片集成)技术 - 灵活性更高 - 良率更高 - 与英特尔制程/工艺完全不同

性能目标 vs 实际: - 目标: 性能提升 40% - 市场质疑: "天方夜谭,很难一步迈这么大" - 实际: 提升了 52%! 🎉

市场表现: - 第一代锐龙 vs 英特尔i7-6900K: - ⚡ 速度更快 - 💰 价格不到一半

🚀 Zen架构的进化

年份 架构 制程 系列 应用
2017 Zen - 锐龙1000 首发
2019 Zen 2 7nm 锐龙3000 PS5, Xbox X/S, Steam Deck
2020 Zen 3 - 锐龙5000 -
2022 Zen 4 5nm 锐龙7000 -

豪赌七纳米: - 2019年下注七纳米制程 - 结果: 一路起飞 🚀

桌面CPU市场份额: - 再次回到与英特尔势均力敌的态势!

🎮 第六章: 游戏业务 - 黑暗中的救命稻草

💡 APU的诞生

2011年 - 推出 APU (Accelerated Processing Unit) - 在一块芯片上融合 CPU + GPU - 当时AMD黑暗时期的救命稻草

苏妈的贡献: - 2012年 加入AMD总负责产品 - 牵头让各大游戏机巨头选择AMD芯片

成果: - 每年带来 2-3亿美元 利润 - 弥补个人电脑端的巨大亏损 - 让AMD活了下来!

当前状态: - 🎯 垄断主机市场 - 按部就班发展即可

🏢 第七章: 数据中心 - 最大增长引擎

🎯 为什么是数据中心?

市场特点: - 客户: 云服务商、超级计算机 - 需求: 高端CPU + GPU - 之前完全是英特尔的天下 - 这是英特尔的命根子!

企业预算分配: - 科技公司约 50% 预算放在服务器上 - 对芯片要求: 高端、高能效比

⚡ EPYC (霄龙) 横空出世

基于Zen架构 → 推出企业级 EPYC芯片

经过几代优化: - 性能超越英特尔至强系列!

💰 TCO优势 - 杀手锏

TCO = Total Cost of Ownership (总拥有成本)

企业关注点: 初期购买成本 (冰山一角) + 持续耗电成本 + 维护成本 + 散热成本 ↓ TCO才是大头!

AMD的"阴险"操作: - 官网提供对比工具 - 可以随意选AMD vs 英特尔芯片对比 - 赤裸裸地拉踩!

对比结果: - 💰 价格: 便宜 20%-50% - 🔋 维护成本: 低 10%-30% - 对于年采购上万台服务器的企业: 省几个亿!

🎤 AMD想说: "随便拉出来一个芯片对比,我又便宜性能又好,耗电量又低,总成本又低!"

合规性: - 英特尔没法告,说明数据真实 - AMD有底气这么做!

📊 市场渗透

主要客户: - Amazon AWS - Microsoft Azure - Google Cloud
- IBM Cloud - 国内互联网企业 (采购比例很高)

😄 "什么时候刷4K视频有点卡了,就说明企业该加购服务器了"

💎 第八章: 收购赛灵思 - 490亿的大手笔

🎯 战略目标

2020年 - 收购芯片巨头 Xilinx (赛灵思)

赛灵思是谁: - FPGA芯片领域头号玩家 - FPGA = 可编程逻辑阵列 - 应用: 电动汽车、火星车、通信基站等

AMD目的: - 大幅扩大业务范围 - 进入新的芯片领域

💡 财经小知识: 换股收购

问题: - 收购价: 490亿美元 - AMD现金: 只有 20-30亿美元

怎么办? → 换股收购!

操作方式: 赛灵思股东 ↓ 换取 1.7234股 AMD股票 ↓ 赛灵思占合并后公司 26%股份

收购价值的神奇变化:

时间点 AMD股价 赛灵思估值
宣布时 基准 350亿美元
完成时 (2022.2.14) +40% 490亿美元 🚀

💰 空手套白狼: - 因为AMD股票狂涨 - 自己的肉变得像"唐僧肉"一样值钱 - 割点肉就能换东西了!

结果: - 芯片史上最大收购 - 收购价在一年多内暴涨 140亿美元!

🤖 第九章: AI时代的爆发 (2023-)

🌊 二级助推

2023 = AI元年

AMD的优势: - 当年收购ATI差点把自己搞死 - 终于迎来丰收! - 市场上唯二的独立显卡公司

🚀 AI芯片表现

产品发布: - 一系列AI芯片推出 - 立即被 Microsoft, Meta, OpenAI 哄抢!

业绩暴涨: - 2023 Q4数据中心业务: 环比暴涨 43% - 达到 23亿美元 - 超过游戏业务 - 成为AMD 第一收入来源!

消费端: - 锐龙系列全部整上 AI引擎NPU - 性能比上一代提升 60%

🔥 AMD现在: All in AI!

📈 股价与市值

首次真正领先: - 虽然英特尔营收、市场份额仍是老大 - 但AMD的潜力和增长势头 - 头一次真正把英特尔甩开了!

⚔️ 第十章: 竞争格局与未来

🆚 AMD vs 英伟达

与英特尔的区别: - 英特尔竞争: 零和博弈 (抢市场、抢客户) - 英伟达竞争: 蓝海市场

市场现状: - AI领域未来3-5年潜在市场: 1500亿美元/年 - 全球AI产能: 只能供应 <20% 需求 - 需求远大于供给!

💬 苏妈: "AMD不用非得击败英伟达,才能在显卡市场取得好成绩"

实际情况: - 只要产品够好,客户大把排队 - "管你是苏妈、老黄、还是隔壁老王,只要好用我都要!" - AMD和英伟达更像是和自己赛跑,和技术赛跑

🔧 英特尔的反击

2021年 - 换新CEO,选择不同路线

战略: - 把宝押在 芯片制造 上 - 狂砸 200亿美元 新建两个晶圆厂 - 目标: 2027年追上台积电工艺

⚠️ AMD和英特尔的战争,还远远没有结束!

💭 第十一章: 深度思考

🎲 芯片行业的本质

表面: - 更新迭代非常快 - 一款明星产品就能扭转颓势

真相: - 每次突破背后都是 至少3-5年甚至更长的投入 - 这是一场豪赌!

⚡ 永不停歇的齿轮

芯片企业的挑战:

  1. 短期: 推动眼前产品的研发、优化、销售
  2. 长期: 判断下一个五年怎么走
  3. 持续: 不间断投入大量成本去押注

与其他行业对比: 其他行业: 一次赌成 → 舒服领先一段时间 ✨ 芯片行业: 再成功 → 领先3-5年 → 新架构出现 → 拍在沙滩上 😱

残酷现实: - 不能吃老本 - 每5年拿出一半身家再去下注 - 豪赌未来5-10年

👴 "看看老黄和苏妈头发花白的速度,就知道得费多少脑子,承受多大压力"

🔄 科技进步的齿轮

芯片领域: - 科技滚动进步的齿轮 - 被淋漓尽致地放大展现

当下的赢家: - 只是暂时的 - 英特尔砸钱建厂 - AMD all in AI + 收购赛灵思 - 这些都是下一场的赌注 - 都是下一轮交锋的序幕
